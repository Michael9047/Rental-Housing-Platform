Flat Markdown-first repository with a single index entry point (`README.md`) that links into four logical areas: design docs (`backend-design.md`, `frontend-design.md`), weekly progress logs (`weekly-report-*.md`), engineering/process guides (`PROJECT_SETUP.md`, `TEAM_DIVISION.md`, `TEAM_WORKFLOW.md`, `map-api-mapping.md`), and archived drafts under `提交/`. The empty `render/` directory is reserved for generated artifacts (e.g., rendered HTML or slides). There is no build system — the module is purely human-readable reference material consumed by contributors and reviewers.
