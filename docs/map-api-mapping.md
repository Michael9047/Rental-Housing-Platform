# 地图找房 — API 按钮映射表

> 基于 backend-design.md 提取，用于前端对接参考  
> 最后更新：2026-06-30

---

## 地图服务 [map] → /map & /geo

| 方法 | 路径 | 说明 | → 前端按钮/页面 |
|------|------|------|----------------|
| POST | /geo/geocode | 地址→经纬度转换 | 发布房源页 → 填写地址后自动获取经纬度 |
| GET | /map/properties | 视口框选返回房源标记列表 | 地图找房页 → 拖拽/缩放地图自动加载标记点 |
| GET | /map/config | 获取地图配置（高德Key+默认中心） | 地图找房页 → 页面初始化获取配置 |

## 周边设施 [poi] → /pois

| 方法 | 路径 | 说明 | → 前端按钮/页面 |
|------|------|------|----------------|
| GET | /pois/{property_id} | 查看房源周边交通/教育/医疗等 | 房源详情页 → "周边设施"标签 → 首次访问自动生成 |

---

> **联调要点**：`/geo/geocode` 虽前缀不同，但与地图找房密切相关——发布房源时自动回填经纬度，地图找房依赖经纬度数据。`/map/properties` 是核心接口，前端防抖 500ms 后调用，Leaflet 手动聚合簇。
